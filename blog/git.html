What are the benefits of version control?

Version control allows you to revert back to an old version of a file if needed (e.g. the changes made broke the program, the old version had a better runtime) or read why a certain change was made.

How does git help you keep track of changes?

Git records all changes made to a file when you commit staged files. You may access the file's change history later on if you wish to do so.

Why use GitHub to store your code?

GitHub is widely used, allows you to access your code anytime and anywhere, easily collaborate with other people, and track issues. Plus it provides a nice viewing platform of everything.
