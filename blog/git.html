How does git help you keep track of changes?

<<<<<<< HEAD <<<<<<< HEAD Git is a version control system (VCS) that records and keeps track of all changes made to a file every time you commit staged files. You may access the file's change history later on if you wish to do so (e.g. read why a certain change was made). By using git, you no longer have to keep copies of old files (e.g. Rough Draft #1, Rough Draft #2, Final), which saves up a lot of space. ======= Git records all changes made to a file when you commit staged files.
